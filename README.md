| Name               | Best     | Average  | Worst    | Memory |
| :----------------- | :------- | :------- | :------- | :----- |
| **Bubble Sort**    | n        | n^2      | n^2      | 1      |
| **Insertion Sort** | n        | n^2      | n^2      | 1      |
| **Selection Sort** | n^2      | n^2      | n^2      | 1      |
| **Quick Sort**     | n log(n) | n log(n) | n^2      | log(n) |
| **Merge Sort**     | n log(n) | n log(n) | n log(n) | n      |
